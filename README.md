## Projects Overview

### Sample Superstore Analysis – SQL
Analyzed the Sample Superstore dataset to uncover business insights:
- Checked data quality and completeness
- Identified top-performing categories, segments, and products
- Detected loss-making products
- Calculated total sales and profit by category, segment, and region

SQL code for this project can be found in `Sql_Project/Superstore_Analysis.sql`.

---

### Video Game Sales Analysis – SQL
Analyzed the global video game sales dataset (`vgsales.csv`) to understand:
- Sales trends by year, genre, and platform
- Total and average sales per game
- Regional market share
- Top-selling games per year

SQL code for this project can be found in `Sql_Project/vgsales_analysis.sql`.

---

### Python Superstore Analysis
Analyzed the **same Sample Superstore dataset** used in SQL to uncover complementary insights:
- Calculated total sales and profit by category, segment, and year
- Analyzed profit distribution and trends
- Visualized data using bar charts, pie charts, and line charts

Python scripts for this project can be found in `Python_Project/Superstore_Analysis.ipynb`.

---

### Power BI Dashboard
Built an interactive Power BI dashboard to visualize sales performance and KPIs:
- Monthly and yearly sales trends
- Performance by category, sub-category, and segment
- Interactive filters for State and Year

Dashboard file can be found in `PowerBI_Project/Sales_Dashboard.pbix`.

---

### Google Sheets Project
Analyzed Spotify streaming data to identify trends, top genres, and song popularity:
- Created Pivot Tables and charts to summarize data
- Evaluated average popularity scores by genre and year
- Highlighted key insights and trends over 2023-2024

Project file can be found in `GoogleSheets_Project/Spotify_Streams_Analysis.xlsx`.
