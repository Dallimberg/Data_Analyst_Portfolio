# Retail Sales Performance Dashboard

## Project Overview
This project analyzes retail sales data to identify performance trends across categories, segments, and regions.
The goal is to provide business insights using interactive dashboards built with Power BI.

## Objective
- Understand sales performance by category and sub-category
- Identify trends across customer segments and regions
- Highlight seasonal trends and sales peaks
- Provide an interactive dashboard for quick business insights

## Solution 
- Imported and connected the Sample Superstore dataset in Power BI
- Built interactive visuals to summarize sales data, including:
  - KPI cards showing Total Sales
  - Monthly sales trend
  - Sales by category and sub-category
  - Orders by customer segment
  - Geographic distribution of sales
- Added interactive filters by State and Year to allow dynamic exploration
- Used SQL queries to clean and aggregate data when necessary

## Tools Used
- Power BI
- SQL

## Key Insights
- Consumer segment represents approximately 51% of total orders.
- Technology category generates the highest sales.
- Sales trend increases toward the end of the year, with a peak in Q4.

## Dashboard Features
- KPI cards showing Total Sales
- Monthly sales trend
- Sales by category and sub-category
- Orders by customer segment
- Geographic distribution of sales
- Interactive filters by State and Year

Sample Superstore dataset.
